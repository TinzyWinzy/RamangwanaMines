# RamangwanaMines

Ramangwana Mining Enterprise Portal — B2B Lead Generation, Project Intake, CRM & Training PWA.

## Stack

- **Frontend:** Vite + React 18 + TypeScript + Tailwind CSS
- **Backend:** Firebase (Firestore, Auth, Storage, Cloud Functions)
- **Payments:** Paynow Zimbabwe
- **Maps:** Leaflet.js
- **Charts:** Recharts

## Getting Started

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and add your Firebase project keys.

## Build

```bash
npm run build
```
